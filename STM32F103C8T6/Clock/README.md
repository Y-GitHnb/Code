# 开发板：
	STM32F103C8T6


# 程序功能：
	系统滴答定时器每秒闪烁板载 LED
	附带串口 1 的空闲中断收发程序


# 原理说明：
	* 按官方配置的滴答定时器 1ms 进一次中断

	* 定时器时间计算：Tout = ((ARR+1) * (PSC+1)) / Tclk
		- ARR（TIM_Period）：自动重装载值，是定时器溢出前的计数值
		- PSC（TIM_Prescaler）：预分频值，是用来降低定时器时钟频率的参数
		- Tclk：定时器的输入时钟频率（单位Mhz），通常为系统时钟频率或者定时器外部时钟频率，STM32F1 主频 72Mhz，STM32F4 主频 168Mhz
		- Tout：定时器溢出时间（单位us），一定注意这个单位是 us
