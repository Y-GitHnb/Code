# 开发板：
	STM32F103C8T6


# 程序功能：
	基于 STM32 单片机万能红外遥控器
	定时器 1 实现微秒延时（delay_us 比 delay_us_2 精准 2-3us）
	定时器 2 通道 2 产生 38KHZ PWM 信号
	定时器 3 通道 1 输入捕获外界红外信号


# 原理说明：
	* STM32cubeMX 生成定时器：
		- 定时器频率 = Tclk / PSC + 1
		- 定时器溢出时间计算方法：Tout = ((ARR+1) * (PSC+1)) / Tclk us
		
	* NEC 数据格式：同步码头、地址码、地址反码、控制码、控制反码。
		- 同步码由一个 9ms 的低电平和一个 4.5ms 的高电平组成
		- 地址码、地址反码、控制码、控制反码均是 8 位数据格式，
			- 数据‘0’ = 560us 低电平 + 560us 高电平
			- 数据‘1’ = 560us 低电平 + 1680us 高电平
		- 按照低位在前，高位在后的顺序发送


# 备注说明：
	* 红外遥控的编码方式目前广泛使用的是 PWM（脉冲宽度调制）的 NEC 协议和 Philips PPM（脉冲位置调制）的 RC-5 协议，代码使用 NEC 协议
		- 8 位地址和 8 位指令长度；
		- 地址和命令 2 次传输（确保可靠性）；
		- PWM 脉冲位置调制，以发射红外载波的占空比代表“0”和“1”；
		- 载波频率 38Khz；
		- 位时间为 1.125ms 或 2.25ms；

	* 定时器 2 发射的信号是和 38KHZ 方波组合的信号
	* 定时器 3 接收的是解码 38KHZ 后的信号
